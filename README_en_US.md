# Chat Pair Plugin for SiYuan

[中文版](./README_zh_CN.md)

A SiYuan plugin for displaying AI conversations with chat bubbles and Q&A pairs.

## Features

- **Chat Bubbles**: Insert chat bubbles with tails pointing left or right
- **Q&A Pairs**: Create paired question and answer bubbles in a single container

## Usage

### Slash Commands

The plugin provides three slash commands:

| Command | Trigger | Description |
|----------|-----------|-------------|
| **Chat Q&A** | `/chat qa` or `/qa` | Insert a pair of Q&A bubbles in a super block |
| **Chat Pair (Right)** | `/chat pair right` or `/对话气泡右` | Insert a single bubble with tail pointing right (Answer) |
| **Chat Pair (Left)** | `/chat pair left` or `/对话气泡左` | Insert a single bubble with tail pointing left (Question) |

### Bubble Styles

- **Right-tail Bubble**: Blue background, used for AI assistant responses
- **Left-tail Bubble**: Gray background, used for user questions
- **Q&A Container**: Pairs both bubbles in a single super block

## Installation

1. Download the plugin from [SiYuan Community Bazaar](https://github.com/siyuan-note/bazaar)
2. Open SiYuan and go to **Settings > Marketplace**
3. Find "Chat Pair Plugin" and click **Install**
4. Restart SiYuan to activate the plugin

## Development

This plugin is built with [Vite](https://vitejs.dev/) and follows the SiYuan plugin development standards.

### Build

```bash
pnpm install
pnpm run build
```

### Development

```bash
pnpm run dev
```
